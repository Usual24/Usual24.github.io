Roooty 버전 히스토리   
============= 
   
Roooty v1.0 — “루티야 안녕?”   
•	주요 기능   
o	메신저로 루티와 바로 대화 가능   
o	앱 설치, 회원가입 필요 없음   
o	학생이 일상·감정·고민을 말하면 루티가 친근한 톤으로 대답   
o	단순한 채팅 기능만 지원   
   
* * *   
   
Roooty v1.1 — “루티야 이거 기억해?”   
•	업데이트 내용   
o	대화 기록을 JSON 형태로 저장 → 안정성 증가   
o	대화 안정성 개선, 오류 감소   
   
* * *   
   
Roooty v1.5 — “루티야 이건 비밀이다?”   
•	업데이트 내용   
o	대화 기록 암호화 저장 → 외부에서 서버를 해킹해도 대화 내용 확인 불가   
o	개인 정보 보호 강화 (실명·거주지 등 미수집)   
o	데이터 보안 업그레이드 완료   
   
* * *   
   
Roooty v2.0 — “루티야 오늘 급식 뭐야?”   
•	새 기능 추가   
o	급식 조회 가능 → “오늘 급식”, “내일 급식”, “9월 18일 급식” 등 자연어로 질문   
o	알레르기 정보 표시 (예: [1. 난류, 2. 우유])   
o	날짜 이해 능력 업그레이드 (오늘/내일/다음 주 등)   
   
* * *   
   
Roooty v2.1 — “루티야 오늘 날씨 어때?”   
•	새 기능 추가   
o	날씨 조회 지원 → “서울 지금 날씨”, “천안 내일 날씨”, “다음 주 수요일 날씨” 등 자연어로 물어보기   
o	온도뿐만 아니라 습도·자외선(UV) 지수, 기상 상태(맑음, 비 등)까지 안내   
o	전국 모든 지역 지원
